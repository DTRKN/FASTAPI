Проект написаный на Python, фрейворк FastAPI
Название проекта Finance Accounting, расчет своих финасов

Для того чтобы посмотреть весь функционал проекта, его нужно запустить, через локальный сервер Uvicorn
Как запуститься сервер, переходим по хосту ('127.0.0.1') и порту ('8000'), и к url адресу добисать ('/docs')
